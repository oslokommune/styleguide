Having the option to trigger content visible is a good way to compress the length of the site you are making. Make sure that if you want the reader to read the information it is smarter to start with the component open and not collapsed.

Triggers get the .osg-collapsible__trigger--expanded modifier when content is expanded.
Triggers should be wrapper in heading elements where possible to help screen readers
