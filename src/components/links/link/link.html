Default

For best practice (Universal design) a link should open in the same window.

- There should be a good reason for not following this rule, (eks opening PDF files) you then also have to use the external link icon.

External

Rule for external link is that it requires an external link icon after the text. Use the external link when you want to link to another website or a new domene.

- Exception on the rule is when we link to sudo domene. Then use default link.

Chevron

Chevron icon left is used for links that will take you back, chevron icon right is used for links that take you forward.
